## Problem · 跟不上的上新节奏

月 GMV $120K 的 Amazon + Shopify 家居品牌, 月上新 80 SKU, 但 3 个兼职写手手写 listing 的速度严重跟不上, 美国站 listing 断档 2 周以上, 新品错过黄金展示期。

**痛点明确**:
- Listing 格式 + 关键词优化耗时（平均每条 45 分钟）
- 德 / 日 / 英三语版本要分别写
- Helium 10 关键词库更新后要回去优化已上架 listing
- 兼职写手质量参差, 返工率高

## Solution · 3 个 Agent 接管内容流水线

Vibecoding Pod V-SEO 场景, 14 天交付:

- **Agent 1 (规格提取)**: 从供应商 PDF + 1688 产品图中抽取技术规格、材质、尺寸, 结构化入 Airtable
- **Agent 2 (多语言生成)**: 调 Claude Sonnet, 按 Amazon listing 模板生成英 / 日 / 德三语版本（标题 + 5 bullet + description + search terms）
- **Agent 3 (关键词优化 + 合规)**: 按 Helium 10 关键词库优化 + 合规检查（避开 Amazon 禁用词如 "best" / "top" / 医疗宣称）
- **数据看板**: 生成条数 / 人工审核通过率 / Amazon 上架后 7 天 sessions 趋势

**关键实现细节**:
- 每条 listing 生成后自动走一遍合规检查, 不通过直接打回重生成
- 产品图同时跑 Google Cloud Vision 提取颜色和场景, 辅助长尾关键词
- 支持批量 Excel 导入 + 批量导出 Helium 10 格式
