## 1. 3 个真 API endpoints 接通

### POST `/api/lead`
- 7 字段输入 · 3 级 validation (字段缺失 / 值非法 / server error)
- 写入 `.shipwright-data/leads.json` (gitignored 本地持久化)
- 返回 `{ id, status: "received", createdAt }` · 201
- LeadForm 切此接口 · localStorage 作保底 backup (API down 时不丢数据)

### GET/PATCH/DELETE `/api/leads`
- `x-admin-token` header / `?token=` query 认证
- GET 返回全量 + count · 401 时 Admin 页显示登录 panel
- PATCH 改 status · 实时同步
- DELETE 单条删除
- Dev 默认 token: `shipwright-dev-token-2026` (env `ADMIN_TOKEN`)

### POST `/api/chat`
- 规则路由 mock AI (10+ 类关键词匹配: 客服 / listing / 广告 / retainer / 价格 / 14 天 / BYOK / 运营 / 内容 / 英文)
- Phase 2 Day 11 替换为真 Claude Haiku streaming
- sessionId 持久化 (localStorage)
- Typing indicator + 错误 fallback (到 /contact)

### E2E 测试通过记录

```

GET /api/leads (no auth)                      HTTP 401
GET /api/leads (with token)                   HTTP 200 · total=1
POST /api/chat                                HTTP 200 · 规则路由正常
```

## 2. Astro hybrid SSR 架构

**从 output="static" → output="server" + `@astrojs/node`**:

- 25 个内容页显式 `export const prerender = true` (批量 sed) · 保持 SEO + 性能
- 3 个 API routes + `/admin/leads` 走 SSR · Node runtime 按需响应
- Build 分两部分: `dist/client/` (静态) + `dist/server/` (Node app)
- 本地 `pnpm preview` 启 Node server 跑完整 SSR (dev 也一样)
- 生产部署 3 选 1:
  - **阿里云 FC** (推荐 国内): Node 20 运行时 · 按调用计费
  - **Cloudflare Pages Functions** (海外): 换 adapter `@astrojs/cloudflare`
  - **Vercel / Netlify** (不推荐国内, 仅 Phase 2 海外)

## 3. Pricing Calculator Island (真交互)

`/pricing` 页底部加 `<PricingCalculator>` 组件. 输入:

- **月 GMV 滑块**: ¥5 万 - ¥1 亿
- **主场景**: V-CS / V-SEO / V-AD / V-CONTENT / V-OPS / Multi (6 选 1)
- **BYOK toggle**: 切换托管 / 自带 Key

输出 (实时):

- 推荐 SKU (Pilot / Vibecoding / Fleet 按 GMV 和场景组合动态推)
- 首月成本 (Setup + Platform [首 3 月免] + AI pass-through)
- 稳态月成本 (Platform + AI)
- 对标自招人力成本

AI cost 按真实模型定价估算 (Claude Sonnet 4.6 · $3/MTok 输入 · $15/MTok 输出 · USD 7 CNY · 托管 10% markup).

**示例**: 月 GMV ¥100 万 · V-CS 客服 · 托管模式 → 推荐 Vibecoding Pod · 首月 ¥59,500 · 稳态 ¥3,900/月. BYOK 切换: 稳态 ¥3,300/月.
